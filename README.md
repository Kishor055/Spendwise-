## 🚀 The Vision: Fintech Noir
In an era of fragmented digital spending, **SpendWise 3.0** centralizes your financial existence. It leverages the **Gemini 2.5 Flash Neural Stack** to decode your "Spending DNA," predict high-fidelity cash flow, and correlate personal habits with global market viability. Our design philosophy, **"Fintech Noir,"** delivers a high-contrast, glassmorphism-driven command center experience that prioritizes data density and physics-based interactions.

## 🧠 Neural Intelligence Modules

### 1. Strategic Wealth Oracle (Co-pilot)
*   **Contextual RAG**: Deep-retrieval of transaction history, budgets, and manifestations for holistic advisory.
*   **Professional Burn Rate**: AI-driven analysis of how your spending impacts your career stability and market readiness.
*   **Neural OCR & Voice**: High-precision receipt extraction using **Gemini Vision** and natural language intent parsing.

### 2. Quantum Pulse Analytics
*   **Predictive Forecasting**: Stochastic modeling of account balances for 7, 30, and 90-day windows.
*   **Subscription Intelligence**: Automatic detection of recurring "leaks" with projected annual loss calculations.
*   **Sector Heatmaps**: High-fidelity Recharts visualizations of sector-wise outflows and efficiency metrics.

### 3. Purchase Decision Engine
*   **Impact Simulation**: AI calculates if a high-value purchase (e.g., MacBook, Real Estate) delays your manifestation goals.
*   **Stochastic Market Logic**: Python-powered Monte Carlo simulations for investment risk and growth.

## 🏗️ System Architecture

SpendWise 3.0 is built on a **Unified Neural Architecture**:

-   **Frontend**: Next.js 15 (App Router) + React 19 (Server Actions).
-   **AI Layer**: Google Genkit v1.0 orchestrating Gemini 2.5 Flash.
-   **Backend**: Firebase (Auth & Firestore) with Zero-Trust Security Rules.
-   **Modeling**: Python Financial Core for advanced statistical simulations.
-   **Visuals**: Framer Motion 12.0 + Recharts 2.x.

## ⚡ Quick Start Terminal

1. **Initialize Matrix**
   ```bash
   git clone https://github.com/Kishor055/Spendwise-.git
   cd Spendwise-
   npm install
   ```

2. **Neural Configuration**
   Create `.env.local` (ensure these are added to your local environment only):
   ```env
   NEXT_PUBLIC_FIREBASE_API_KEY=your_production_key
   NEXT_PUBLIC_FIREBASE_PROJECT_ID=your_project_id
   GOOGLE_GENAI_API_KEY=your_gemini_pro_key
   ```

3. **Launch Engine**
   ```bash
   npm run dev
   ```
